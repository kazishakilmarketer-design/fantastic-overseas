# Login Credentials - Fantastic Overseas Portal

This document lists the default credentials for the Fantastic Overseas Rebranded portal dashboards (Admin & Agent sections).

---

## 1. Super Administrator Portal
Used to manage recruitment packages, add/delete agents, edit About/Message descriptions, manage the gallery, and approve new candidate registrations.

* **Login URL (Public):** `https://fantasticoverseas.com/login/admin.html`
* **Login URL (Local Dev):** `http://localhost:8080/login/admin.html`
* **Default Credentials:**
  * **Email:** `admin@fantasticoverseas.com`
  * **Password:** `admin`
  * **Default Name:** `Super Admin`

---

## 2. Agents Portal
Used by agency representatives to enlist new candidate files, update applicant visa status steps, checklist document submissions, and update transaction billing.

* **Login URL (Public):** `https://fantasticoverseas.com/agent.html`
